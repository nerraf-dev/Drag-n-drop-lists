Drag 'n' Drop Generator

Getting Started:

To create a drag and drop activity you have two options:

  1. Fill in the form below
  2. Upload a template file

Option 1:

  • Fill in the game name
  • Enter the category names
  • Enter the items for each category. ONE per line.
  • Press the "+ Category" button to add extra categories
  • Press "Submit"

Option 2: Load from file

  • Click "Choose File"
  • Upload a suitable template file
  • The fields are filled in ready for you to check
  • Press "Submit"
Template file format:
                    {
                        "gameName": "Word types",
                        "categories": [
                          {
                            "name": "Adjectives",
                            "items": [
                              "red",
                              "tall"
                            ]
                          },
                          {
                            "name": "Nouns",
                            "items": [
                              "table",
                              "Paris",
                            ]
                          }
                        ]
                      }
